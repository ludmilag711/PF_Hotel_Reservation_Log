**Date:** 1st March 2026
**Version** 1

## Project goals
Simple Python app to log, retrieve, edit and keep track of hotel reservations. For the first version these basic functions are all it must be able to do. 

## Features
- **F1:** Add reservations
- **F2:** Be able to view all resevations in table format
- **F3:** Be able to cancel/modify reservation 
- **F4:** Be able to search for specific reservations

## Reservation fields
Field/ Required or optional?

reservation_ID - required
first_name - required
last_name - required
checkin_date - required
checkout_date - required
num_nights - required
num_adults - required
num_children - required
room_type - required
room_num - required
guest_type - required
comments - optional

## Constraints

This version aims to achieve the core functions of a reservation log and as such is likely to not yet be able to:
- Take payments
- Generate graphs
- Take a credit card guarantee and encrypt the details for security
- Check for room category availability
